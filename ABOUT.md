# About myRAG

## 📖 项目背景

myRAG 项目源于企业知识管理的实际需求。在日常工作中，我们积累了大量的文档资料（技术文档、项目报告、培训材料等），但传统的关键词搜索方式难以快速定位所需信息，更无法实现智能问答。

基于 RAG（检索增强生成）技术，我们开发了这套系统，让文档"活"起来，能够通过自然对话的方式获取知识。

## 🎯 核心愿景

**让知识触手可及** - 通过 AI 技术降低知识获取门槛，让每个人都能快速找到所需信息。

## 💡 技术选型理由

| 技术 | 选择理由 |
|------|---------|
| **Go 后端** | 高性能、并发处理能力强、部署简单 |
| **React 前端** | 生态成熟、组件丰富、开发效率高 |
| **Qdrant** | 开源向量数据库、支持混合检索、性能优秀 |
| **NATS** | 轻量级消息队列、适合异步任务处理 |
| **MinIO** | S3 兼容、可自部署、成本低 |
